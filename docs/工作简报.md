## 📊 项目概览

本项目实现了论文中的部署区域分解算法（Algorithm 1）和基于分解变换的多目标粒子群优化算法（MOPSO-DT），用于解决复杂区域内的多功能雷达网络部署优化问题。

### 核心特性
- ✅ 区域分解：将复杂多边形分解为凸子区域并分配二进制编码
- ✅ MOPSO-DT：混合变量（连续+二进制）多目标优化
- ✅ 坐标变换：归一化坐标到物理坐标的映射
- ✅ 性能优化：Numba JIT 加速核心计算
- ✅ 完善测试：26个单元测试 + 性能基准测试

### 快速开始

```bash
pytest tests/test_mopso_pytest.py -v

# 运行性能测试
python tests/test_performance.py

# 运行示例
python examples/demo_decomposition.py
```

### MOPSO-DT 优化示例
```python
from src.mopso import MOPSO_DT

# 定义评估函数
def evaluate(Phi):
    # Phi: 决策变量矩阵 (J, 2+N_bin)
    coverage = np.mean(Phi[:, 0])
    interference = np.std(Phi[:, 1])
    return np.array([coverage, interference])

# 创建优化器（启用性能优化）
mopso = MOPSO_DT(
    J=5,              # 5个雷达节点
    N_bin=3,          # 3位区域编码
    evaluate_func=evaluate,
    N_P=50,           # 50个粒子
    T_max=200,        # 200次迭代
    use_batch_update=True  # 启用批量更新优化
)

# 执行优化
archive, stats = mopso.optimize()

# 获取Pareto前沿
cont, binary, objs = mopso.get_pareto_front()
print(f"找到 {len(objs)} 个非劣解")
```
