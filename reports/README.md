# Reports Directory

This directory stores ARIS-generated audits and review outputs tied to the updated project state.

## Expected artifacts

- `EXPERIMENT_AUDIT.md`
- `EXPERIMENT_AUDIT.json`
- `PAPER_CLAIM_AUDIT.md`
- `PAPER_CLAIM_AUDIT.json`
- `CITATION_AUDIT.json`

## Rule

Reports in this directory should refer to the updated evidence in `results/` or the final migrated thesis in the Tongji template, not to the legacy `paper/` draft alone.
